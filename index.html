The 0 Party

Taking you from 0 to Hero!

Our Mission

To bust the myth that IT is a rapidly-changing industry. Look around you--there's still plenty of C, FORTRAN, and COBOL code. Mark Zuckerberg supposedly still uses Emacs. Genius CS professors still make very spartan websites. Aren't you tired of company recruiters telling you that you need to be a quick learner? What if you could graduate college (we don't believe in coding bootcamps here) knowing you'd never have to learn another programming language? What if your employment options were only determined by what you learned the past four years?

Our Values

  • Using Vim or Emacs for everything, even this website.
  • Using spaces instead of tabs, even if Vim tries to autodent our HTML
  • Learning only 3-5 technologies, especially if they're all from the same company
  • Diagramming everything with UML; again, even this website.
